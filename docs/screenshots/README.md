## 截圖規範建議

- **解析度**：Retina 1x 即可（不用到 2x，檔案會太大）
- **瀏覽器視窗**：寬度建議 1440px，能完整顯示 sidebar + 畫布 + Detail Panel
- **背景**：使用系統暗色主題（配合工具本身深色 UI）
- **馬賽克**：如有個人資訊或機密 domain 用語，請模糊處理
- **檔名**：嚴格按編號命名（`NN-name.png`），方便 README 引用
